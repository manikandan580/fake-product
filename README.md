# AI Powered Fake Product & Review Detection System

This is a **Flask-based web application** that detects fake products and fake reviews on e-commerce platforms. Users can upload products, view all products, and check for fake or real products and reviews using basic AI/ML logic.

---

## ✅ Features

- Upload product with image, title, price, description, features, brand, address, reviews, rating, and link.
- View all products in a Flipkart-style UI.
- Product Detection: Check if a product is fake or real based on price, rating, and description.
- Review Detection: Detect fake and real reviews and show percentage.
- Dataset stored in CSV (`products.csv` and `reviews.csv`).
- Images stored in `static/uploads/`.
